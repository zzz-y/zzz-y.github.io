AJAX是什么

发表于 2019-03-21 | 更新于 2019-03-24 | 分类于 js

AJAX 即 “Asynchronous JavaScript and XML”（异步的 JavaScript 与 XML 技术），AJAX的概念由Jesse James Garrett 提出。

AJAX 的使用方法：

  1. 使用 XMLHttpRequest 发请求
  2. 服务器返回 XML 格式的字符串
  3. JS 解析 XML，并更新局部页面
